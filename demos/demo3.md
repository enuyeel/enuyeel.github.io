**Image Processing Demo #4**

<br>

Built with: <u>C++, GLSL, OpenCV, OpenGL, GLEW, GLM, SDL2, ImGui, MSVC</u>

<br>

This application demonstrates the following post-processing algorithms:

<br>

*-* Negative
*-* Sobel Operator
*-* Chromatic Aberration
*-* Pixellation
*-* Afterimage
*-* Meshify
*-* Gradient
*-* Sketch
